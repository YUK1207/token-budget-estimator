Four modules, one preflight decision.

Most agent waste happens before the agent understands the task. This skill turns the first move into a measured decision instead of a blind repo scan.

01

Token Forecast

Predicts a full task range, including file reads, command output, failed tests, retries, analysis, and final response.

02

ROI Assessor

Scores whether the task should execute now, split first, start with discovery, or wait for a better scope.

03

Budget Contract

Sets hard working boundaries: files, commands, log lines, denied actions, and stop-loss checkpoints.

04

Context Diet

Plans what Codex should read first, what comes second, and what should stay out of context until needed.

A contract before execution.

The output is not just a number. It is a working agreement that tells Codex how to spend context and when to stop.

Token forecast8k-144k, extreme risk
ROIdiscovery_first, high value but high cost pressure
Budget rulesRead 6 files, run 2 commands, summarize logs under 80 lines
Stop-lossStop if the fix crosses more than 3 modules or risk gets worse

Context diet, not context hoarding.

A good first pass does not read everything. It reads the smallest useful set, then escalates only when the evidence says so.

first

Failing test file, directly referenced module, test configuration.

second

Caller and callee modules, fixture setup, recent related changes.

avoid

Unrelated UI pages, documentation, build artifacts, vendor and generated folders.

Use it from Codex or the CLI.

The skill is installable, but the underlying script is just as useful in terminals, automation, and local experiments.

controlpython scripts/estimate.py control --task "Fix failing tests" --cwd . --budget 50000 --mode cheap
estimatepython scripts/estimate.py estimate --task "Upload this skill to GitHub" --cwd . --context-window 128000
comparepython scripts/estimate.py compare --task "Explain a function" --task "Refactor the project" --cwd .
optimizepython scripts/estimate.py optimize --task "Make this broad request safe for Codex" --budget 50000
