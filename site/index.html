The release lineup

Four decisions before one expensive run.

The page is intentionally product-first: it explains why the skill matters, not how to install or operate it.

01 / Forecast

Token Forecast

Predicts a realistic interval for the full agent path, including reading code, running commands, failed tests, logs, retries, and final response.

02 / ROI

ROI Assessor

Decides whether the task is worth executing now, should begin with discovery, or should be split before it burns through context.

03 / Contract

Budget Contract

Frames execution boundaries around file reads, command runs, log output, retry policy, and stop conditions before work begins.

04 / Diet

Context Diet

Points Codex toward the highest-value context first, and names what should be skipped until evidence says it matters.

Product moment

A preflight layer for serious agent work.

The launch narrative is simple: if an agent can spend context, it should be able to justify that spend before execution.

ROI before action. Agent execution becomes a budgeted decision, not a blind run.
STEP 01 Estimate the burn range Low, expected, and high token exposure.
STEP 02 Classify the risk Task type, repo size, test uncertainty, log blast radius.
STEP 03 Set execution boundaries Readable files, runnable commands, output caps, stop-loss.
STEP 04 Choose the highest-value path Start narrow, verify early, expand only when evidence pays.
